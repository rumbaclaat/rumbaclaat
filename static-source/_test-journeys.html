DEV HUB · NOT FOR PRODUCTION

Test journeys

Direct links to every form, journey, and success state in the site. Use this for QA, walk-throughs, and showing flows to stakeholders without filling forms in by hand.

Dev bypass is active on every page. That means:

  • The age gate is skipped automatically.
  • All form fields are pre-filled with plausible test data on page load.
  • All required attributes are stripped — you can submit any form by clicking its button, no typing needed.
  • The cart is seeded with 1× Original Reserve if empty, so the cart page renders populated.

To return to production behaviour, remove the line <script>window.RC_DEV_BYPASS = true;</script> from every HTML file.
